# طريقة رفع المشروع على GitHub Pages

1. ادخل على https://github.com وسجل دخولك.
2. اضغط زر + أعلى الصفحة، ثم New repository.
3. اكتب اسم المستودع: ubt-sports-platform
4. خله Public.
5. اضغط Create repository.
6. اضغط Add file ثم Upload files.
7. ارفع كل الملفات الموجودة داخل مجلد المشروع، وأهم شيء index.html.
8. اضغط Commit changes.
9. ادخل Settings من صفحة المستودع.
10. من القائمة اليسار اختر Pages.
11. عند Source اختر Deploy from a branch.
12. عند Branch اختر main و /root ثم Save.
13. انتظر دقيقة إلى دقيقتين.
14. سيظهر الرابط بهذا الشكل:
   https://USERNAME.github.io/ubt-sports-platform/

استبدل USERNAME باسم حسابك في GitHub.
